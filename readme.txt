GenomProj v 0.2

Реализовано:
1. Чтение любого текстового файла. Файл выбирается в диалоговом окне.
2. Сохранение изображение на диск в формате jpeg или bmp. Сохранить изображение можно только после того, как оно построено, или построение изображения прервано.
3. Изображение строится быстрее. Сделано за счет того, что не выводится каждый пиксель, а выводиться конечное изображение.
4. Появилась возможно прервать построение изображения кнопкой "Отмена". После нажатия на нее, на экран будет выведено изображение, которое было сформировано.

Порядок работы:
1. Запустить файл GenomProj.exe
2. Нажать кнопку "открыть файл". В диалоговом окне выбрать текстовый файл с кодом генома.
3. Задать масштаб. Маштаб должен быть положительным, целым числом. В поле ввода масштаба вводить только цифры.
4. Задать ширину изображения в пикселях. Если ввести в поле ввода "1", то будет построено изображение с соотношением сторон 3:4.
5. Нажать кнопку "ОК".
6. Ждать. Прогресс построения будет отображать в индикаторе выполнения (progress bar).
7.1 Изображение появилось. Просматривать изображения можно с помощью горизонтальной и вертикальной полос прокрутки (scrollbar).
7.2. Можно прервать построение изображения, нажав кнопку "Отмена".
8. Сохранение изображения - кнопка "Сохранить".
